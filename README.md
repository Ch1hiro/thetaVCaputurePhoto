# thetaVCaputurePhoto

## 目的
RICOH Thetaによる輝度画像合成用の写真撮影が目的です。
お手持ちのコンピュータとWiFi経由で接続したTheta Vに対し、1時間おきに自動で輝度画像合成画像を撮影するためのプロジェクトです。

### 輝度画像の合成に必要な画像
文献[1]の表2を参考にISO感度、シャッタースピード、色温度を以下の11種類に決定しました。

 表2 推奨（高精度）版 11枚 撮影条件([1]より抜粋・編集)

| ISO感度  | シャッタースピード                     | WB（色温度）                   |
|----------|----------------------------------------|-------------------------------|
| 100      | 1/25000                                | 5200K                         |
| 100      | 1/12500                                | 5200K                         |
| 100      | 1/3200                                 | 5200K                         |
| 100      | 1/1600                                 | 5200K                         |
| 100      | 1/400                                  | 5200K                         |
| 100      | 1/200                                  | 5200K                         |
| 100      | 1/50                                   | 5200K                         |
| 100      | 1/25                                   | 5200K                         |
| 100      | 1/6                                    | 5200K                         |
| 100      | 1/3                                    | 5200K                         |
